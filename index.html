About me

Hello! My name is Juan and I enjoy building Web Apps and Websites for people. My path in the world of programming began in 2014 when I was just 12 years old, my first steps began with C, C++ and then with PHP and JavaScript, it wasn't until 2019 when I started Web Development.

In that year I became interested in Wordpress, where I had the opportunity to participate in some projects, being the most interesting for me Facilito E-Commerce. After 2020 I started to build VillaDonq with Laravel, since that year I dedicated myself to improve my technique and skills to provide the best service and great works!

I've been working with:

  • Laravel
  • Wordpress
  • VueJs
  • Elementor
  • Oxygen Builder
  • Bootstrap
  • Tailwind

When I started with this framework I loved it, i worked with its versions 7, 8, and 9 using from Laravel UI with Bootstrap to Vue Js, I feel comfortable with its Ecosystem using tools like Breeze or Jetstream and its ORM Eloquent. I used this to build: VillaDonq, Maucake CMS and API

For me Wordpress is the perfect tool when a client just wants a solid and attractive Web Site, i have experience building web sites with builders like Elementor and Oxygen, or if you want i can build customs templates with ACF just for you. I used this to build: Facilito E-Commerce
